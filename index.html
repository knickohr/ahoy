AhoyDTU

Eine Open-Source Firmware, um Hoymiles ® Wechselrichter aller Generationen auszulesen

lizensiert unter CC-BY-NC-SA 4.0

Projekt auf Github: https://github.com/lumapu/ahoy/

Wir haben eine Firmware programmiert, die es ermöglicht Hoymiles ® Wechselrichter der Generationen MI, HM, HMS und HMT auszulesen. Das Ganze geschieht lokal und ohne Cloud. Die DTU verbindet sich per WLAN und stellt die Daten live auf einer übersichtlichen Weboberfläche im (lokalen) Netzwerk bereit.

Durch MqTT und eine REST API lässt sich die AhoyDTU sehr leicht in ein bestehendes Smart-Home System integrieren. Die AhoyDTU wurde bereits erfolgreich in ioBroker und Home-Assistant eingebunden. Die Daten können z.B. mit Grafana visualisiert werden. Über das Smart-Home System kann auch eine Nulleinspeisung erfolgen.

Auf der Weboberfläche können sehr leicht Wechselrichter eingerichtet werden. Für jedes angeschlossene Panel (Ann. ein Panel pro Eingang) wird Spannung, Strom, Leistung und Ertrag dargestellt, dies macht eine Fehlersuche einfach oder ermöglicht ein Monitoring der gesamten Anlage. Die AhoyDTU eignet sich für die Optimierung der Ausrichtung (z.B. Winkel, Abschattung) einzelner Module sehr gut. Eine AhoyDTU kann je nach Ausbaustufe bis zu 16 Wechselrichter verwalten - alle Generation mit einem Gerät.

Neuigkeiten

Dezember 2023: Grid Profile kann gelesen werden

Ahoy hat sich zur Aufgabe gemacht, ein Tool zu sein, die eigene Solaranlage zu überwachen und zu analysieren. Um noch tiefer in die Materie einzusteigen können jetzt auch die auf dem Wechselrichter hinterlegten Grid Profiles ausgelesen werden.

Diese Profile geben Auskunft darüber, wie sich der Wechselrichter in bestimmten Situationen verhält, z.B. ab welcher Netzspannung er abschaltet oder aber welcher Netzfrequenz er eine Synchronisation startet.

November 2023: HMS-xxxxW Serie

Leider kann Ahoy die Wechselrichter der HMS-xxxxW Serie (noch) nicht auslesen. Diese Wechselrichter haben eine integrierte DTU und kein Funkmodul mehr wie die drei vorherigen Generationen.

Juni 2023: HMS / HMT Preview Firmware veröffentlicht

Ahoy unterstützt mit dem ESP32 und einem Funkmodul (CMT2300A) jetzt auch die HMS und HMT Serie.

Alle Inverter der HMS / HMT Serie werden zusätzlich unterstützt (hier nur eine Auswahl):
HMS-350, HMS-500, HMS-800, HMS-1000, HMS-1600, HMS-1800, HMS-2000, HMT-1600, HMT-1800, HMT-2250

Download der Preview-Firmware: Development Firmware (Hinweis: die Firmware kann uU. Fehler enthalten)

Visualisierung

Lasse alle Werte deines Hoymiles ® Wechselrichters ohne Cloudanbindung auslesen.
Sowohl die AC-Parameter (grüner Hintergrund) als auch die DC-Parameter (Panele, blauer Hintergrund) werden ausgelesen. Die Visualisierung schafft eine schnelle Übersicht über den Zustand des / der Inverter.

Hardware

Die Hardware besteht im wesentlichen nur aus zwei Komponenten: Einem Wifi-Modul ESP32 (wahlweise ESP8266) und einem Funkmodul NRF24L01+ (oder CMT2300A für HMS / HMT)
Die Hardwarekosten belaufen sich auf ca. 15€ und werden damit zu den günstigsten Teilen der PV Anlage zählen. Die Komponenten können sehr einfach zusammengebaut werden (löten) und ohne Programmierkenntnisse über unser Online-Tool mit der Firmware versehen werden.

Verbindungen

AhoyDTU unterstützt MqTT und hat zusätzlich eine REST API. Im Fall von MqTT werden alle ermittelten Werte, wahlweise im Sekundentakt, an den hinterlegten (in den Einstellungen gesetzt) MqTT-Broker gesendet.
